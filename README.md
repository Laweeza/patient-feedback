# Patient Feedback

The purpose of this program is to collect patient feedback after their doctor appointments.

[Installation](#installation) •
[Getting Started](#getting-started) •
[Technologies](#technologies) •

## Installation

```
npm install
```

## Getting Started

```
docker-compose up
```

## Run Tests

```
npm run test
```

```
npm run cypress:run
```
