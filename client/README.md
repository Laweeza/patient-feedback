## Installation

To install depedencies, run:

```
npm install
```

## Development

To use the dev server, use:

```
npm start
```

## Run Tests

To run unit tests, run:

```
npm run test
```

With the server running, to run integration tests use:

```
npm run test:system
```

## Environment Variables

| Variable           | Value    |
| ------------------ | -------- |
| REACT_APP_BASE_URL | host url |
